 Who can use

App is usefull for students, bachelors or any people who are living in shared house, where every individual in a house has to maintain database of what all expenditures he/she has done on shared resources and settle the amount on equal basis at certain intervals of time(for eg, monthly, querterly, etc.,)

 How to use
  1. Registration
    • Any one individual from the group has to create a database where all expenditures are stored
    • Then he has to share the database name with all the group members
    • Now every group individual has to register for the application(For registration database name is required)
  2. Login
    • Now member can login to the application using login credentials(credentials include database name)
  3. Use
    • Now can add the expenditure
    • Can see other members expenditures
    • And some more operations are provided
